## 5-level paging activation

Enable when `CPUID.7.ECX.LA57=1`, fall back to 4-level. Implementation detail; v1 supports both with runtime detection.

## TSC sync at boot

v1 lean = master-broadcast + cross-check. v2 may switch to per-CPU `cpuid+rdtscp` rendezvous if drift requires.

## FSRM for kernel `memcpy`

v1 lean = yes, when `CPUID` bit set. Benchmark before committing.

## KASLR seed source

`RDRAND` + boot timer. Deferred to v1.x; v1 ships a fixed kernel image (no KASLR yet).
